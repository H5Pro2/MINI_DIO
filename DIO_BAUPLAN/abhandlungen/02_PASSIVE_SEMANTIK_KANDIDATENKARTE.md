# Abhandlung: Passive Semantik-Kandidatenkarte

Stand: 2026-06-06

Diese Abhandlung beschreibt den nächsten fachlichen Schritt nach den bisherigen Mini-DIO-Erkenntnissen. Sie ist keine aktive Handlungslogik und keine Strategie. Sie beschreibt, wie wiederkehrende passive Fragmente geordnet werden können, ohne sie zu früh als Reife, Richtung oder Entry zu behandeln.

## 1. Ausgangslage

Mini-DIO hat bis zum aktuellen Stand passive Fragmentwiederkehr sichtbar gemacht.

Wiederkehrender Kern:

```text
sehen_form_stability
fuehlen_mcm_coherence
```

Wiederkehrende Eigenverdichtung:

```text
fuehlen_mcm_tension
hoeren_energy_tone
reflection_context_alignment
reflection_context_carry
reflection_context_strain
```

Diese Fragmente sind nicht zufällig als einzelne Werte zu lesen. Sie tauchen in mehreren passiven Nähe-Kontexten auf. Damit entsteht eine Vorform innerer Semantik.

Die Gefahr liegt darin, diese Wiederkehr zu früh aktiv zu machen.

Ungültige Lesung:

```text
Dieses Fragment kehrt wieder.
Also ist es ein Signal.
Also darf DIO handeln.
```

Gültige Lesung:

```text
Dieses Fragment kehrt wieder.
Es ist ein Kandidat innerer Bedeutung.
Es muss weiter passiv geordnet, geprüft und gegen Rauschen abgegrenzt werden.
```

## 2. Ziel der Karte

Die passive Semantik-Kandidatenkarte soll wiederkehrende Fragmente in Bedeutungsschichten ordnen.

Sie beantwortet nicht:

```text
Soll DIO handeln?
```

Sie beantwortet:

```text
Welche Art von innerer Spur liegt vor?
```

Die Karte trennt:

```text
sensorischer Kern
affektive Feldwirkung
Reflexionsanteile
Randfragmente
Rauschen
```

Damit entsteht eine Brücke zwischen technischer Diagnose und neurologischer Deutung.

## 3. Schichten der Semantik-Karte

### 3.1 Sensorischer Kern

Der sensorische Kern beschreibt stabile Wahrnehmungsfragmente.

Aktueller Kandidat:

```text
sehen_form_stability
```

Mögliche Lesung:

```text
Eine äußere Form bleibt über Zeit erkennbar.
Sie zerfällt nicht sofort.
Sie kann als Wahrnehmungskern gehalten werden.
```

Grenze:

```text
Sensorischer Kern ist keine Richtung.
Sensorischer Kern ist kein Entry.
Sensorischer Kern ist nur erkennbare Formnähe.
```

### 3.2 Affektive Feldwirkung

Die affektive Feldwirkung beschreibt, wie eine Wahrnehmung im MCM-Feld wirkt.

Aktuelle Kandidaten:

```text
fuehlen_mcm_coherence
fuehlen_mcm_tension
```

Mögliche Lesung:

```text
Eine wahrgenommene Form erzeugt Kohärenz oder Spannung im Feld.
Das Feld erlebt nicht nur Daten, sondern Wirkung.
```

Grenze:

```text
Feldwirkung ist keine Strategie.
Feldwirkung ist innere Lage.
```

### 3.3 Akustisch-energetischer Anteil

Der akustisch-energetische Anteil beschreibt Energie, Frequenz und Spannung der Außenwelt.

Aktueller Kandidat:

```text
hoeren_energy_tone
```

Mögliche Lesung:

```text
Die Welt wird nicht nur gesehen, sondern auch energetisch gehört.
Der Ton ist kein Entry-Anker, sondern ein Stimulus der inneren Wahrnehmung.
```

Grenze:

```text
Hören ist keine direkte Handlung.
Hören ist energetische Wahrnehmung.
```

### 3.4 Reflexionsanteile

Reflexionsanteile zeigen, dass eine Wahrnehmung nicht nur wirkt, sondern in einen inneren Kontext gerät.

Aktuelle Kandidaten:

```text
reflection_context_alignment
reflection_context_carry
reflection_context_strain
```

Mögliche Lesung:

```text
Die wahrgenommene Lage wird innerlich eingeordnet.
Sie trägt, belastet oder passt zu einem Kontext.
```

Grenze:

```text
Reflexion ist keine Handlung.
Reflexion ist die Distanzierung von Reiz und Innenlage.
```

### 3.5 Randfragmente

Randfragmente sind Fragmente, die sichtbar werden, aber noch keine stabile Bedeutung tragen.

Mögliche Lesung:

```text
Ein Fragment taucht auf,
aber seine Wiederkehr ist noch schwach,
uneindeutig oder nur in wenigen Kontexten sichtbar.
```

Grenze:

```text
Randfragment ist Beobachtung.
Randfragment ist keine Semantikreife.
```

### 3.6 Rauschen

Rauschen beschreibt Fragmentspuren ohne ausreichende Wiederkehr oder ohne sinnvolle Kopplung.

Mögliche Lesung:

```text
Das System hat etwas registriert,
aber es ist noch keine tragende innere Spur.
```

Grenze:

```text
Rauschen darf nicht erzwungen gedeutet werden.
Rauschen darf zerfallen.
```

## 4. Neurologische Deutung

Neurologisch betrachtet entspricht diese Karte einer Vorordnung präbewusster Aktivität.

Ein Gehirn verarbeitet nicht alles sofort bewusst. Es bindet Reize zunächst in Schichten:

```text
sensorische Stabilität
Körper-/Feldwirkung
affektive Bewertung
Kontextnähe
Nachhall
Gedächtnisspur
```

Die passive Semantik-Kandidatenkarte ist daher keine Denkschicht im aktiven Sinn. Sie ist eher eine Karte der Vorbedeutung.

Formuliert als Prozess:

```text
Ich sehe etwas.
Es wirkt in mir.
Es kehrt wieder.
Es bekommt eine innere Spur.
Diese Spur ist noch kein Entschluss.
```

## 5. MCM-Deutung

Im MCM-Sinn ist die Semantik-Karte ein Versuch, Feldverdichtung geordnet sichtbar zu machen.

Das MCM-Feld trägt nicht nur Spannung. Es kann auch wiederkehrende Spannungsformen halten.

Hypothetische MCM-Folge:

```text
Wahrnehmung
-> Feldwirkung
-> Nachhall
-> Wiederkehr
-> Fragmentinsel
-> Semantik-Kandidat
```

Damit wird MCM nicht nur als Spannungsraum betrachtet, sondern als Raum möglicher innerer Bedeutungsbildung.

Diese Aussage bleibt hypothetisch:

```text
Die Semantik-Karte beweist keine Intelligenz.
Sie zeigt nur, welche passiven Feldfragmente wiederkehrend geordnet werden können.
```

## 6. Technische Grenze

Die Karte darf nicht aktiv gelesen werden.

Erlaubt:

```text
passive Diagnose
Strukturierung
Vergleich über Läufe
Trennung von Kern, Rand und Rauschen
```

Nicht erlaubt:

```text
Runtime-Lesung
Entry
Richtung
Gate
Motorik
Strategie
```

Die technische Grenze lautet:

```text
Semantik-Kandidat ist kein Handlungskandidat.
```

## 7. Warum dieser Schritt wichtig ist

Ohne diese Karte besteht die Gefahr, dass jedes wiederkehrende Fragment sofort überinterpretiert wird.

Mit dieser Karte entsteht ein kontrollierter Zwischenraum:

```text
Nicht mehr reines Rauschen.
Noch keine aktive Reife.
Aber sichtbare Vorbedeutung.
```

Das ist für Mini-DIO wichtig, weil es den Weg zwischen Wahrnehmung und Handlung entschleunigt.

Die Zielrichtung:

```text
Wahrnehmung darf wachsen.
Semantik darf entstehen.
Handlung bleibt getrennt,
bis reale Konsequenz und Reife sie tragen.
```

## 8. Nächster hypothetischer Schritt

Der nächste technische Schritt wäre ein passives Diagnosemodul, das die vorhandenen Fragmentwiederkehr-Daten in diese Schichten einordnet.

Möglicher Output:

```text
semantic_candidate_layer
fragment
recurrence_state
source_count
meaning_candidate
boundary
```

Beispielhafte Zuordnung:

```text
sehen_form_stability
-> sensorischer Kern

fuehlen_mcm_coherence
-> affektive Feldwirkung / Kohärenzkern

hoeren_energy_tone
-> akustisch-energetischer Anteil

reflection_context_alignment
-> Reflexionsanteil
```

Auch dieses Modul müsste passiv bleiben.

## 9. Zusammenfassung

Die passive Semantik-Kandidatenkarte ist der nächste logische Schritt nach der Fragmentwiederkehr.

Sie hält fest:

```text
Wiederkehrende Fragmente sind nicht automatisch Handlung.
Sie sind Kandidaten innerer Bedeutung.
```

Im neurologischen Sinn:

```text
präbewusste Aktivität wird vorgeordnet.
```

Im MCM-Sinn:

```text
Feldverdichtung wird als mögliche Vorbedeutung sichtbar.
```

Im technischen Sinn:

```text
Mini-DIO bekommt eine saubere passive Zwischenebene
zwischen Rauschen und gereifter Handlung.
```
