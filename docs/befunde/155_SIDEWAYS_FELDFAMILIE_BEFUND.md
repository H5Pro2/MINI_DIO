# Sideways-Feldfamilie Als MCM-Befund

Stand: 2026-06-18

## Zweck

Diese Datei dokumentiert die erste gezielte Seitwaertspruefung nach der Expansionsachse.

Geprueft wurde:

```text
data\kontrolliert_2024_moderate_sideways_test1_1000_5m_SOLUSDT.csv
```

Ziel war nicht, eine neue Regel oder Handlung abzuleiten.

Ziel war:

```text
Liest MINI_DIO Seitwaertsdruck als eigene Feldfamilie,
oder faellt diese Welt in eine bereits bekannte MCM-Innenfeldklasse?
```

## Ergebnis

Die Seitwaertswelt bildet keine harte neue Klasse.

Sie faellt in die bekannte:

```text
ruhige Naehegruppe
```

und zeigt gleichzeitig eine deutliche innere Begleitung durch:

```text
tragend_unruhig
```

## Interpretation

Seitwaertsdruck erscheint in diesem Lauf nicht als chaotischer oder zerfallender Zustand.

MINI_DIO liest die Welt als getragen, aber nicht vollkommen ruhig:

```text
Die Welt bleibt nahe genug am Feldzentrum,
erzeugt aber fortlaufende kleine Unruhe.
```

Damit ist Seitwaertsbewegung fuer MINI_DIO nicht einfach Stillstand.

Sie wirkt eher wie:

```text
ruhige Naehe mit Oberflaechenbewegung
```

Das passt zur bisherigen MCM-Lesung:

```text
Nicht die Richtung allein entscheidet ueber die Feldklasse,
sondern die Belastbarkeit der Bewegung im Innenfeld.
```

## Feldzeit

Die Feldzeitdiagnose zeigt:

```text
temporal_first_contact: 919
temporal_far_return: 74
temporal_immediate_afterimage: 1
```

Das bedeutet:

```text
Seitwaertsdruck erzeugt hier wenig tiefen Nachhall.
Die Welt wird stabil gelesen, aber ueberwiegend als neue Gegenwart,
nicht als stark wiederkehrende Feldzeit.
```

Wichtig:

```text
Stabilitaet bedeutet hier nicht automatisch tiefe Erinnerung.
```

## Forschungswert

Dieser Lauf erweitert die MCM-Feldordnung um eine wichtige Unterscheidung:

```text
Expansion kann ruhig oder belastet sein.
Stress kann Gegenpolwirkung erzeugen.
Seitwaerts kann getragen sein, ohne tiefe Feldzeit auszubilden.
```

Damit wird die bisherige Topologie nicht widerlegt.

Sie wird feiner:

```text
Es gibt stabile Zentrumsnaehe,
angespannte Uebergangsgruppen,
Stress-Gegenpole
und oberflaechennah getragene Seitwaertslagen.
```

## Wie Es Weitergeht

Als naechstes sollte eine zweite Seitwaertswelt geprueft werden.

Hierarchie:

1. Grundfrage: Bleibt Seitwaertsdruck in der ruhigen Naehegruppe?
2. Unterpruefung: Entsteht bei anderer Seitwaertswelt mehr Feldzeit oder mehr Unruhe?
3. Folgeschritt: Wenn beide Seitwaertswelten aehnlich reagieren, kann `oberflaechennah_getragen` als passive Leseklasse dokumentiert werden.
