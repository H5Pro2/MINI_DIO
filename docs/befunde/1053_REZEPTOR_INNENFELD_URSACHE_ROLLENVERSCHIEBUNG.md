# 1053 - Rezeptor-/Innenfeld-Ursache der Rollenverschiebung

## Grundfrage

Welche vorgelagerte Bedingung erklärt die Rollenverschiebung aus 1051 und 1052?

## Einordnung

Die direkte Rohkerzen-/Rohwelt-Zuordnung pro Weltgruppe liegt in den aktuellen Befunddateien nicht vollständig vor.

Diese Prüfung liest deshalb nicht die äußere Ursache direkt, sondern die nächstliegende innere Vorstufe:

```text
Rollenbedingung -> Rezeptor-/Innenfeldwirkung -> Feldverschiebung
```

Erzeugte Datei:

- `1053_REZEPTOR_INNENFELD_URSACHE_ROLLENVERSCHIEBUNG.csv`

## Ergebnis

Die Verschiebung wirkt nicht wie ein neues Feldsystem.

Sie wirkt wie eine veränderte Aufnahme- und Bindungsqualität:

```text
Belastung wird später nicht harmloser.
Sie wird seltener dominant ausgelöst.

Rekopplung wird später nicht massiver.
Sie wird sauberer gebunden.
```

## Befunde

### Belastete Fragmentierung

`netz_fragmentiert_belastet`

- früh: 3123 Beobachtungen
- spät: 1627 Beobachtungen
- Delta: -1496

Rekopplung bleibt ähnlich negativ, Strain bleibt ähnlich positiv.

Lesung:

```text
Der Belastungsraum bleibt qualitativ belastet,
wird aber später deutlich seltener aktiviert.
```

Das spricht nicht für weniger Belastungsfähigkeit, sondern für weniger dominante Auslösung.

### Zentrum mit Anschluss

`netz_zentrum_mit_anschluss`

- früh: 6819 Beobachtungen
- spät: 5634 Beobachtungen
- Rekopplung steigt: 0.0066 -> 0.0231
- Nachbarschaft sinkt: 5.625 -> 5.0

Lesung:

```text
Zentrum wird weniger überladen,
aber rekoppelt besser.
```

Das wirkt wie selektivere Zentrumsnähe.

### Rekoppelnde Verbindung

`netz_rekoppelnd_verbunden`

- Knoten: 28 -> 30
- Beobachtungen: 499 -> 299
- Nachbarschaft: 1.607 -> 1.833
- Strain: -0.0128 -> -0.0162

Lesung:

```text
Rekopplung wird weniger breit,
aber sauberer nachbarschaftlich gebunden und entlastender.
```

Das passt zu 1051, wo `rekoppelnd -> rekoppelnd` später von Rang 13 auf Rang 3 steigt.

### Getragene Drift

`netz_driftend_getragen`

- Knoten: 22 -> 33
- Rekopplung: 0.0244 -> 0.0480
- Nachbarschaft: 2.182 -> 0.667

Lesung:

```text
Drift verteilt sich später auf mehr Knoten,
ist weniger dicht vernetzt,
aber feldseitig rekoppelnder.
```

Das wirkt wie fragmentärere, aber tragendere Drift.

### Einzelrekopplung

`netz_rekoppelnd_einzeln`

- Knoten: 17 -> 26
- Beobachtungen: 34 -> 85

Lesung:

```text
Später entstehen mehr kleine Entlastungspunkte ohne tragendes Netz.
```

Das kann eine Vorform von späterer Rollenbindung sein, ist aber noch keine stabile Netzrolle.

## Interpretation

Die bisherige Diagnose lässt folgende Lesung zu:

```text
Frühe Gruppe:
mehr dichte Belastung,
mehr offene Kippnähe,
stärkere Fragmentierungsdominanz.

Späte Gruppe:
weniger Belastungsaktivierung,
selektivere Zentrumsnähe,
mehr rekoppelnde Selbstbindung,
mehr kleine Entlastungspunkte.
```

Das ist keine harte Regulation von außen.

Es wirkt wie passive Innenfeldorganisation:

```text
Das Feld verschiebt nicht die Bedeutung der Belastung.
Es verschiebt deren Dominanz und Bindungsform.
```

## Forschungswert

Diese Prüfung ergänzt 1052:

- 1052 zeigte die interne Rollenverschiebung.
- 1053 zeigt die vorgelagerte Rezeptor-/Innenfeldlesung.

Damit entsteht eine sauberere Kette:

```text
Weltkontakt
-> rezeptorische Aufnahme
-> Innenfeldwirkung
-> Rollenaktivierung
-> Rollenübergang
```

Die Rohweltursache ist damit noch nicht bewiesen, aber die interne Bedingungsseite ist präziser eingegrenzt.

## Grenze

Diese Datei ist keine Rohweltanalyse.

Sie kann noch nicht beantworten:

```text
Welche konkrete Kerzenstruktur löst die spätere Rekopplung aus?
Welche Ton-/Energieform reduziert Fragmentierungsdominanz?
Welche visuelle Form begünstigt Zentrum ohne Überladung?
```

Das bleibt der nächste Schritt.

## Nächster Prüfpunkt

Als nächstes muss die Rohweltseite rückgelesen werden.

Benötigt wird eine Zuordnung:

```text
Weltgruppe
-> verwendete Datensequenzen
-> Rohweltmerkmale
-> Rezeptorachsen
-> Rollenverschiebung
```

Erst dann kann sauber gesagt werden, welche äußere Weltspannung die beobachtete MCM-Feldverschiebung begünstigt.
