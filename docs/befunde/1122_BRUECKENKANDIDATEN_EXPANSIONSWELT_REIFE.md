# 1122 - Brueckenkandidaten in einer Expansionswelt

Diese Synthese ist passiv. Sie erzeugt keine Handlung, kein Gate, kein Entry-Signal und keine Richtungslogik.

## Frage

Bleibt `dio_00ly` auch in einer anderen Weltklasse sichtbar, oder war die bisherige Reifung nur an die vorherigen Folgewelten gebunden?

## Gepruefte Welt

- `kontrolliert_2023_positive_expansion_test1_1000_5m_SOLUSDT.csv`
- 1000 Kerzen
- 2 passive Laeufe mit frischer Memory

## Befund

| Familie | Ereignisse | Rolle | Feld | Lesart |
|---|---:|---|---|---|
| `dio_00ly` | 6 | tragende Verarbeitung | 4 offen, 2 rekoppelt | weiter sichtbar, aber schmaler |
| `dio_0pq6` | 0 | - | - | nicht sichtbar |

## Technische Deutung

`dio_00ly` bleibt in der Expansionswelt sichtbar, verliert aber Breite.

Die Ereignisse zeigen weiterhin:

```text
tragende Verarbeitung
reduzierter Strain
geordnete Hoerhaltung
```

Gleichzeitig liegt die Feldbindung haeufiger bei `offen` als bei `rekoppelt`. Das spricht nicht gegen die Familie, sondern gegen eine starre Deutung:

```text
dio_00ly ist kein festes Symbol;
dio_00ly ist eine tragende Brueckenrolle,
deren Feldqualitaet von der Weltklasse mitbestimmt wird.
```

`dio_0pq6` bleibt auch hier aus. Damit verfestigt sich die Lesart als lokaler Kontakt-/Umschaltkandidat.

## Bedeutung fuer das Bedeutungsnetz

Die Expansionswelt trennt Reife von Starrheit:

- Reife heisst nicht, dass ein Zeichen immer gleich stark erscheint.
- Reife heisst, dass eine Rolle bei veraenderter Weltklasse wiedererkennbar bleibt.
- Wenn die Weltklasse anders ist, darf die Rolle schmaler, offener oder weniger rekoppelt auftreten.

Damit wird `dio_00ly` belastbarer, aber zugleich genauer begrenzt:

```text
tragend wiederkehrend,
aber nicht global dominant.
```

## Grenze

Die Expansionswelt ist nur eine 1000er Stichprobe. Fuer eine stabile Reifebeschreibung braucht es eine zweite Expansionswelt oder eine ruhige Seitwaertswelt als Gegenklasse.

## Wie es weitergeht

Als naechstes sollte eine ruhige Seitwaertswelt geprueft werden. Wenn `dio_00ly` dort ausbleibt oder anders kippt, ist die Familie wahrscheinlich an aktive Bewegungsraeume gebunden. Wenn sie dort tragend bleibt, wird sie zu einem deutlich breiteren Brueckenrollen-Kandidaten.
