# Passive Regulationswahrnehmung

Stand: 2026-06-20

## Zweck

Diese Notiz formuliert eine kleine passive Regulationswahrnehmung aus den Milieu- und Driftbefunden.

Sie ist keine aktive Regulation:

```text
keine Handlung
kein Gate
keine Strategie
keine harte Schwelle
```

Sie beschreibt nur, wie MINI_DIO eine innere Feldbewegung als Wahrnehmungsqualität lesen kann.

## Hierarchie

1. Grundfrage: Kann MINI_DIO unterschiedliche Tragqualitäten einer Feldbewegung passiv lesen?
2. Unterprüfung: Milieu-Drift aus 338-342 in einfache Innenfeldqualitäten übersetzen.
3. Folgeschritt: diese Qualitäten später gegen weitere Welten prüfen, ohne daraus Handlung abzuleiten.

## Vier passive Qualitäten

### Eng Getragen

Eine Feldbewegung erscheint in engem, wiederkehrendem Milieu.

Lesart:

```text
Die Bewegung bleibt nah an einer stabilen Innenfeldlage.
Das Kontaktmilieu trägt ähnlich.
Die Feldbewegung driftet wenig.
```

Das ist keine Handlungsfreigabe. Es ist nur eine Wahrnehmung von innerer Tragfähigkeit.

### Offen Driftend

Eine Feldbewegung bleibt erkennbar, aber ihr Milieu ist offener.

Lesart:

```text
Die Bewegung ist vorhanden.
Das Milieu ist nicht eng gebunden.
Die Feldlage bleibt eher offen als stark rekoppelnd.
```

Das passt zu Welten, in denen die Passung niedriger ist oder die Feldbewegung nicht in eine enge rekoppelnde Lage fällt.

### Breit Driftend

Eine Feldbewegung bleibt wiederkehrend, aber sie erscheint in mehreren Milieus.

Lesart:

```text
Die Bewegung selbst bleibt lesbar.
Das Kontaktmilieu variiert.
Die Bedeutung ist nicht zerfallen, aber breiter getragen.
```

Der aktuelle Befund liest `1t5bcxp -> 183drjy` in Langwelten so:

```text
breit driftend
```

Das ist fachlich wichtig: Wiederkehr muss nicht starr sein. Eine Bedeutung kann als Bewegung stabil bleiben, während ihr Milieu driftet.

### Fragmentiert

Eine Feldbewegung erscheint in vielen unterschiedlichen Milieus ohne klare dominante Tragqualität.

Lesart:

```text
Die Bewegung kommt vor.
Die Milieus streuen stark.
Die Innenfeldqualität ist nicht klar gebunden.
```

Der aktuelle Befund liest `183drjy -> 1t5bcxp` in Langwelten so:

```text
stark fragmentiert
```

## Bezug Zu 338-342

Die Befundreihe zeigt:

- Kurzfenster: `1t5bcxp -> 183drjy` wirkt deutlich rekoppelnd.
- Langwelten: dieselbe Bewegung bleibt häufig, aber ihr Milieu wird breiter.
- `183drjy -> 1t5bcxp` ist deutlich stärker fragmentiert.
- Weltmerkmale erklären Milieu-Neigungen, erzwingen aber keine Feldbewegung.

Damit entsteht eine passive Regulationswahrnehmung:

```text
Ich lese nicht nur, dass eine Feldbewegung vorkommt.
Ich lese auch, wie sie getragen wird.
```

## Abgrenzung

Diese Schicht darf nicht als Regel verstanden werden.

Falsch wäre:

```text
eng getragen -> handeln
fragmentiert -> blockieren
```

Richtig ist:

```text
eng getragen -> passive Innenfeldqualität
fragmentiert -> passive Innenfeldqualität
```

MINI_DIO beobachtet damit die Qualität seiner eigenen Feldbewegung.

## Wie es weitergeht

Als nächstes sollte geprüft werden, ob diese vier Qualitäten über weitere Weltgruppen reproduzierbar bleiben.
Wenn ja, können sie als stabile passive Innenfeldwahrnehmung in den MINI_DIO-Kern übernommen werden.
