# MCM-Synthese: Rollenwechsel statt Feldverlust

## Zweck

Diese Synthese verbindet 956, 958 und 959.
Sie klaert den scheinbaren Widerspruch:

```text
Brueckenlesung: Reifungstokens nicht wiedergefunden.
Nicht-Bruecken-Lesung: dieselben Tokens bleiben ueberwiegend sichtbar.
```

## Befund

Die siebte XRP-Weltgruppe bildet nach bisheriger Lesung keine Brueckenlandschaft:

- 0 Brueckenkanten,
- 0 Brueckenkerne,
- 0 Anschlussanker.

Trotzdem bleiben die meisten Reifungstokens in anderer Feldform sichtbar:

| Token | Vorherige Langzeitklasse | Nicht-Bruecken-Lesung | Arbeitsdeutung |
|---|---|---|---|
| `0z748ck` | langfristig getragen | zentrumsnah getragen | Reifung bleibt, Rolle wechselt |
| `1hdpu9s` | langfristig belastet getragen | zentrumsnah getragen | belastete Reifung bleibt sichtbar |
| `0om13wf` | weltabhaengig getragen | Rekopplungsfeld | weltabhaengige Reifung bleibt als Feldform sichtbar |
| `00yl137` | abgeschwaecht oder verloren | Randspannung | nicht stabil, aber als Randspur vorhanden |
| `00dz86x` | kurzfristige Oberflaeche | offen sichtbar | Oberflaeche reorganisiert sich |
| `00nzcuc` | kurzfristige Oberflaeche | nicht sichtbar | echte Unsichtbarkeit in dieser Lesung |
| `0db07p4` | verschwunden bestaetigt | zentrum schwach | Bruecken-Verschwinden war kein vollstaendiger Feldverlust |
| `1ahj81f` | verschwunden bestaetigt | zentrumsnah getragen | Bruecken-Verschwinden war kein vollstaendiger Feldverlust |

## Kernerkenntnis

Ein Zeichen kann seine Rolle verlieren, ohne als Feldinformation zu verschwinden.

Das ist fuer die MCM wichtig:

```text
Rolle != Existenz
Bruecke != Bedeutung
Nicht-Bruecke != Chaos
```

Die siebte Welt zeigt nicht einfach einen Zusammenbruch der bisherigen Memory.
Sie zeigt eine Umlagerung:

```text
Brueckenordnung faellt weg.
Zentrums-, Rekopplungs-, Rand- und Oberflaechenordnung bleiben.
```

## Bedeutung fuer MINI_DIO

MINI_DIO darf spaeter nicht nur fragen:

```text
Ist mein bekanntes Zeichen wieder in derselben Rolle sichtbar?
```

Sondern:

```text
Ist mein bekanntes Zeichen noch als Feldform sichtbar,
auch wenn seine Rolle gewechselt hat?
```

Damit entsteht eine bessere Memory-Qualitaet:

- weniger vorschnelles Verwerfen,
- weniger starres Auswendiglernen,
- bessere Trennung zwischen Rollenverlust und Bedeutungsverlust,
- stabilere Lesung von Weltwechseln.

## Wissenschaftliche Arbeitsdeutung

Die beobachtete Reifung wirkt nicht wie ein statischer Speicher.
Sie wirkt wie eine topologische Feldspur, die je nach Weltspannung unterschiedliche Ausdrucksformen annimmt:

- Bruecke,
- Zentrum,
- Rekopplungsfeld,
- Randspannung,
- offene Oberflaeche,
- echte Unsichtbarkeit.

Das passt zur bisherigen MCM-Arbeit:
Eine Bedeutung ist nicht nur ein gespeicherter Wert, sondern eine im Feld wieder lesbare Organisationsform.

## Wie es weitergeht

Als naechstes sollte eine Rollenwechsel-Memory entstehen.
Sie speichert passiv:

```text
Dieses Zeichen war Bruecke.
In anderer Welt wurde es Zentrum / Rand / Rekopplung / Oberflaeche.
```

Nicht als Handlung, nicht als Gate, sondern als bessere MCM-Feldbiografie eines Zeichens.
