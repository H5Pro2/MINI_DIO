# Befund 244 - Mehrwelt-Feld-Episoden-Vorschau und lokale Syntax

Stand: 2026-06-19

## Ausgangspunkt

Befund 241 zeigte auf einer einzelnen kontrollierten Welt:

```text
Lokale Rekopplungsfenster und lokale Kippfenster tragen bereits entstehende Feld-Episoden-Vorschau-Syntax.
```

Die nächste Grundfrage war:

```text
Bleibt diese Vorschau-Syntax weltbezogen stabil,
oder bildet jede Welt eigene lokale Feld-Episoden-Sprache?
```

## Prüfwelt

Es wurden drei frische passive Preview-Welten verglichen:

```text
PREVIEW_REAL_2023
PREVIEW_NEG_2023
PREVIEW_EXP_2023
```

Alle drei Welten wurden mit je zwei passiven Läufen geprüft.
Die Diagnosen liegen hier:

```text
242_MEHRWELT_LOKALE_MULTISENSORISCHE_KOPPLUNG_PREVIEW_DIAGNOSE.md
243_MEHRWELT_FELD_EPISODEN_PREVIEW_SYNTAX_DIAGNOSE.md
```

## Ergebnis

Die Mehrwelt-Diagnose zeigt keine beliebige Zerstreuung.
Es entsteht eine gemeinsame Grundsprache mit weltbezogenen Varianten.

Das dominante Feldsymbol ist weltübergreifend:

```text
dio_mcm_episode_02xikfk
```

Es erscheint besonders stark in lokalen Rekopplungsfenstern:

```text
PREVIEW_EXP_2023  lokal_rekoppelnd  dio_mcm_episode_02xikfk  11 Fenster
PREVIEW_NEG_2023  lokal_rekoppelnd  dio_mcm_episode_02xikfk  10 Fenster
PREVIEW_REAL_2023 lokal_rekoppelnd  dio_mcm_episode_02xikfk   9 Fenster
```

Damit wirkt `dio_mcm_episode_02xikfk` wie eine gemeinsame lokale Rekopplungsnähe-Sprache.

Gleichzeitig gibt es weltbezogene Erweiterungen.
In der Expansionswelt tritt zusätzlich auf:

```text
dio_mcm_episode_1t5bcxp
```

Dieses Symbol erscheint in Expansions- und Negativwelt, aber nicht als dominante Grundform der Realwelt.

Lokale Kippnähe zeigt ebenfalls `dio_mcm_episode_02xikfk`, aber weniger eindeutig.
Dazu treten stärker welt- oder abschnittsbezogene Kippvarianten auf:

```text
dio_mcm_episode_1eik02d
dio_mcm_episode_0e9ekzq
dio_mcm_episode_0eje6op
dio_mcm_episode_182yyt2
dio_mcm_episode_037i64j
```

## Lesart

Die derzeitige Lesart ist:

```text
Lokale Rekopplung bildet eine gemeinsame Vorschau-Grundsprache.
Lokale Kippnähe bleibt stärker varianten- und weltabhängig.
```

Das ist fachlich wichtig.
Wenn ein Symbol in mehreren Welten wiederkehrt, spricht das für eine stabile Innenfeldform.
Wenn ein Symbol nur in bestimmten Welten oder Rollen auftritt, spricht das für situative Feldbedeutung.

Damit entsteht eine erste Unterscheidung:

```text
gemeinsame Feldsprache
  -> wiederkehrende Rekopplungsnähe

weltbezogene Feldsprache
  -> spezifische Kipp-, Spannungs- oder Expansionslage
```

## Grenze

Das ist keine Handlung und keine Strategie.

Die Vorschau-Symbole dürfen nicht als Gate, Entry, Motorik oder Richtung gelesen werden.
Sie zeigen nur, dass das Innenfeld während des Weltkontakts lokale Feld-Episoden-Lagen ausbildet.

Auch `dio_mcm_episode_02xikfk` ist kein festes Zielbild.
Es ist ein beobachtetes, wiederkehrendes Preview-Symbol innerhalb der geprüften Welten.

## Forschungswert

Der Befund stärkt die Annahme:

```text
MINI_DIO speichert nicht nur neue Einzelzeichen.
Es bildet eine wiederkehrende lokale Feldsprache,
die über mehrere Welten teilweise stabil bleibt und sich je Welt erweitert.
```

Das passt zur bisherigen MCM-Linie:

- wiederkehrende Innenfeldlagen verdichten sich,
- stabile Nähe kann zentrumsnah oder rekoppelnd wirken,
- Kippnähe bleibt beweglicher,
- Weltkontakt erzeugt Varianten, aber nicht völlige Beliebigkeit.

## Wie es weitergeht

Als nächstes wird die gemeinsame Grundsprache isoliert.

Grundfrage:

```text
Was trägt dio_mcm_episode_02xikfk in mehreren Welten wirklich?
```

Unterprüfung:

```text
Rekopplungsfenster mit dio_mcm_episode_02xikfk gegen Kippfenster mit demselben Symbol vergleichen.
```

Folgeschritt:

```text
Wenn dasselbe Feldsymbol je Rolle andere Wirkung trägt,
dann ist nicht nur das Symbol wichtig,
sondern seine Kopplung an Rolle, Weltspannung und Innenfeldzustand.
```
