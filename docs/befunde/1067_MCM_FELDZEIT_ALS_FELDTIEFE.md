# 1067 - MCM-Feldzeit als Feldtiefe

Diese Abhandlung haelt eine aktuelle MINI_DIO-Erkenntnis fest:

```text
Bedeutung entsteht nicht nur durch Wiederholung.
Bedeutung entsteht durch Wiederholung mit zeitlicher Feldwirkung.
```

## Ausgangspunkt

MINI_DIO zeigt bisher, dass Zeit im MCM-Feld nicht nur als aeussere Ablaufachse gelesen werden sollte.

Die bisherige README-Grenze bleibt gueltig:

```text
Nachhall ist Kontext, nicht Kontrolle.
Feldzeit ist Integrationsqualitaet, keine hart programmierte Zeitachse.
```

Daraus folgt eine praezisere Lesart:

```text
Zeit ist nicht nur Reihenfolge.
Zeit ist die Art, wie eine Weltlage im Feld Tiefe gewinnt.
```

## Kernbegriff

MCM-Feldzeit bezeichnet hier nicht Uhrzeit, Takt oder reine Sequenznummer.

MCM-Feldzeit bezeichnet die im Feld entstehende zeitliche Qualitaet einer wiederkehrenden Weltlage.

Sie kann mehrere Formen annehmen:

```text
frisch
nachhallend
verblassend
wiederkehrend
driftend
rekoppelnd
```

Diese Formen beschreiben keine harte Klassifikation.

Sie beschreiben, wie stark eine Feldlage noch wirkt, wie sie zurueckkehrt, ob sie driftet, ob sie sich abschwaecht oder ob sie erneut an bestehende Rollen anschliesst.

## Abgrenzung zu linearer Zeit

Lineare Zeit fragt:

```text
Wann trat ein Ereignis auf?
Wie oft trat es auf?
Wie weit liegt es zurueck?
```

MCM-Feldzeit fragt:

```text
Wie wirkt dieses Ereignis noch im Feld?
Welche Rolle beruehrt es erneut?
Ist es neu, nachhallend, verblassend oder rekoppelnd?
Verschiebt es eine Bedeutungsinsel?
Stabilisiert es Zentrum, Bruecke, Rand oder Drift?
```

Damit wird Feldzeit nicht als Messachse vor das Feld gestellt.

Sie entsteht aus der Feldwirkung selbst.

## Frisch

Eine frische Feldlage ist eine Weltwirkung, die erstmals oder ohne erkennbare starke Vorbindung auftritt.

Sie ist noch nicht tief in eine Bedeutungsfamilie integriert.

Sie kann auffaellig sein, aber ihre Rolle ist noch offen.

```text
frisch = neue Wirkung ohne gesicherte Rollenbindung
```

Forschungsrelevant ist hier nicht nur die Intensitaet.

Entscheidend ist, ob diese frische Lage spaeter wiederkehrt und dann eine Rolle bildet.

## Nachhallend

Eine nachhallende Feldlage ist nicht mehr nur aktueller Reiz.

Sie traegt eine Vorwirkung mit sich.

Sie steht im Sequenzkontext und kann dadurch staerker oder klarer gelesen werden als ein isoliertes Ereignis.

```text
nachhallend = aktuelle Feldwirkung plus wirksamer Sequenzkontext
```

Wichtig bleibt:

```text
Nachhall ist Kontext, nicht Kontrolle.
```

Nachhall darf also nicht als versteckter Steuermechanismus gelesen werden.

Er erklaert nur, warum eine spaetere Feldlage anders wirkt als dieselbe Lage ohne Vorgeschichte.

## Verblassend

Eine verblassende Feldlage verliert Feldbindung.

Sie ist nicht geloescht, aber sie traegt weniger Spannung, weniger Rekopplung oder weniger topologische Naehe.

```text
verblassend = sinkende Feldwirkung ohne vollstaendige Aufloesung
```

Das ist wichtig, weil Bedeutung in MINI_DIO nicht nur durch Anwesenheit entsteht.

Auch Abschwaechung ist Information.

Eine Rolle kann dadurch an Randnaehe gewinnen, an Brueckenfunktion verlieren oder in offene Drift uebergehen.

## Wiederkehrend

Eine wiederkehrende Feldlage tritt erneut auf und beruehrt erkennbare vorhandene Struktur.

Sie ist mehr als Wiederholung eines Zeichens.

Sie ist Wiederholung mit Feldbezug.

```text
wiederkehrend = erneute Weltwirkung mit Anschluss an bekannte Feldlage
```

Hier liegt eine zentrale MINI_DIO-Erkenntnis:

```text
Nicht das einzelne Zeichen allein ist entscheidend.
Entscheidend ist die wiederkehrende Feldrolle.
```

Wiederkehr wird dadurch zur Bedingung von Bedeutungsverdichtung.

## Driftend

Eine driftende Feldlage bleibt nicht stabil an derselben Rolle gebunden.

Sie verschiebt Naehe, Spannung, Randlage oder Brueckenbezug.

```text
driftend = wiedererkennbare Wirkung mit veraenderter Feldposition
```

Drift ist kein Fehler.

Drift zeigt, dass Bedeutung im Feld nicht statisch gespeichert wird.

Sie wird unter neuer Weltspannung neu verortet.

Damit kann dieselbe Bedeutungsfamilie je nach Weltlage naeher am Zentrum, naeher am Rand oder staerker als Uebergang erscheinen.

## Rekoppelnd

Eine rekoppelnde Feldlage verbindet aktuelle Wirkung mit vorhandener Innenordnung.

Sie stabilisiert, bindet oder fuehrt eine Lage in bekannte Rollennaehe zurueck.

```text
rekoppelnd = erneute Anbindung an tragende Feldordnung
```

Besonders wichtig ist die Unterscheidung:

```text
Rekopplung ist keine Wiederholung allein.
Rekopplung ist Wiederholung mit tragender Rueckbindung.
```

Das erklaert, warum bestimmte `dio_*`-Formen nicht nur haeufig auftreten, sondern als Rollen erscheinen.

Sie wiederholen nicht nur Zeichen.

Sie wiederholen eine Feldfunktion.

## Bedeutung als zeitliche Feldverdichtung

Die bisherige MINI_DIO-Lesung kann damit zugespitzt werden:

```text
Weltkontakt
  -> Sinnesaufnahme
  -> Rezeptorhaltung
  -> MCM-Feldwirkung
  -> Feldzeit
  -> Rollenbindung
  -> Bedeutungsverdichtung
```

Bedeutung entsteht nicht direkt aus Rohdaten.

Bedeutung entsteht auch nicht automatisch aus Haeufigkeit.

Bedeutung entsteht, wenn eine Weltlage im Feld wiederkehrt, nachhallt, driftet, verblasst oder rekoppelt und dadurch eine Rolle ausbildet.

## Konsequenz fuer MINI_DIO

MINI_DIO sollte Feldzeit daher nicht nur als Protokollfolge behandeln.

Zu pruefen ist nicht nur:

```text
Wie oft erscheint eine Form?
```

Sondern:

```text
In welchem Feldzustand erscheint sie?
Welche Vorwirkung traegt sie?
Welche Rolle beruehrt sie erneut?
Wird sie staerker, schwaecher, randnaeher, zentrumsnaeher oder brueckenhafter?
```

Damit wird Feldzeit eine eigene Forschungsschicht.

Sie liegt zwischen Sequenz und Bedeutung.

## Methodische Vorsicht

Diese Abhandlung ist kein Beweis der MCM-Theorie.

Sie ist eine praezisierte Lesart der bisherigen MINI_DIO-Befunde.

Die Grenze bleibt:

```text
Feldzeit wird beobachtet.
Feldzeit wird nicht als fertige Form erzwungen.
```

Wenn eine zeitliche Qualitaet nur durch Benennung entsteht, ist sie kein Befund.

Ein Befund liegt erst vor, wenn sich Unterschiede in Nachhall, Verblassen, Drift, Wiederkehr oder Rekopplung wiederholt in der Feldorganisation zeigen.

## Forschungsfragen

Die naechste Forschung sollte pruefen:

```text
Wann wird ein frischer Reiz zur wiederkehrenden Rolle?
Wann wird Nachhall zur stabilen Bedeutungsnaehe?
Wann verblasst eine Rolle ohne vollstaendige Aufloesung?
Wann ist Drift offene Suche und wann beginnende Reorganisation?
Wann ist Rekopplung nur Wiederholung und wann tragende Rueckbindung?
```

Besonders wichtig ist die Frage:

```text
Welche `dio_*`-Formen tragen nicht nur Haeufigkeit,
sondern feldzeitliche Tiefe?
```

## Schluss

MINI_DIO zeigt damit eine wichtige Erweiterung des Zeitbegriffs.

Zeit ist im MCM-Feld nicht nur Ablauf.

Zeit ist Feldtiefe.

Sie zeigt sich als Frische, Nachhall, Verblassen, Wiederkehr, Drift und Rekopplung.

Die zentrale Erkenntnis lautet:

```text
Bedeutung entsteht nicht nur durch Wiederholung.
Bedeutung entsteht durch Wiederholung mit zeitlicher Feldwirkung.
```

Damit wird MCM-Feldzeit zu einer Schluesselschicht zwischen Weltkontakt, Innenfeldordnung und Bedeutungsverdichtung.
