# Late-Positive-Expansionswelt-Befund

Stand: 2026-06-18

## Zweck

Nach positiver Expansion, extremer Expansion und Recovery-Expansion wurde eine Late-Positive-Welt geprüft.

Die Leitfrage:

```text
Bildet late positive eine Übergangsform
zwischen ruhig_feldzeitnah und last_feldzeitnah?
```

## Geprüfte Welt

```text
data/kontrolliert_2024_late_positive_test1_1000_5m_SOLUSDT.csv
```

Die Welt wurde aus der vorhandenen 10k-Late-Positive-Welt als 1000-Zeilen-Testfenster erstellt.

## Befund

Late Positive bildet tatsächlich eine Übergangsform.

Im Feldklassen-Report wird sie als:

```text
angespannte Übergangsgruppe
```

gelesen.

In der relativen Kurzsegment-Lesung wird sie als:

```text
last_feldzeitnah
```

gelesen.

Damit liegt sie nicht so belastet wie extreme Expansion, aber auch nicht so ruhig wie positive oder Recovery-Expansion.

## Interpretation

Late Positive zeigt eine bewegte Feldzeit mit spürbarer Restlast.

Das Feld trägt die Welt noch überwiegend:

```text
field_carried: 963
field_strained: 31
```

Aber die dominante Wirkung ist nicht mehr `stabil`, sondern:

```text
tragend_unruhig
```

Das ist entscheidend:

```text
Das Feld kollabiert nicht,
aber es wird unruhiger getragen.
```

## Bedeutung Für Die MCM-Lesung

Die Expansionsachse hat jetzt vier lesbare Stufen:

```text
Positive Expansion
  -> ruhig_feldzeitnah

Extreme Expansion
  -> lastnah / last_feldzeitnah

Recovery Expansion
  -> Rückkehr zu ruhig_feldzeitnah mit Restspannung

Late Positive
  -> last_feldzeitnah als Übergangsform
```

Damit entsteht eine feinere organische MCM-Lesung:

```text
Nicht jede Bewegung ist Last.
Nicht jede Erholung ist reine Ruhe.
Nicht jede Last ist Kollaps.
```

MINI_DIO scheint zu unterscheiden:

- stabile getragene Bewegung,
- unruhig getragene Bewegung,
- lastnahe Feldzeit,
- rekoppelnde Erholung.

## Grenze

Der Befund gilt zunächst für ein 1000-Zeilen-Fenster.

Eine größere Aussage braucht weitere Late-Positive-Fenster oder eine volle 10k-Prüfung.

## Wie Es Weitergeht

Als nächstes sollte die Expansionsachse zusammengefasst werden.

Ziel:

- eine klare Übersicht der vier Expansionsformen,
- Wertevergleich,
- MCM-Deutung,
- offene Prüfpunkte.
