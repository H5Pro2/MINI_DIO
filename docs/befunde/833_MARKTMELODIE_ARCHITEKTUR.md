# 833 - Marktmelodie Architektur

## Grundidee

MINI_DIO kann eine Aussenwelt nicht nur visuell und feldbezogen lesen, sondern auch tonal.
Ein Kursverlauf wird dabei als bewegte Weltspur verstanden, aus der eine passive Hoerwahrnehmung entstehen kann.

Das Ziel ist keine Musik im kuenstlerischen Sinn und keine menschliche Sprache.
Das Ziel ist eine technische Hoerform:

```text
Weltbewegung -> Tonhoehe / Lautstaerke / Rauigkeit / Rhythmus -> Tonrolle -> Melodiephrase -> DIO-Klangtoken
```

## Trennung

Die Marktmelodie ist eine rezeptorische Vorstufe.
Sie wirkt nicht direkt als Handlung, Gate oder Strategie.

Sie darf spaeter gegen MCM-Feldlagen gelegt werden, aber sie ersetzt das MCM-Feld nicht.

Saubere Trennung:

- Sehen liest Form und Struktur.
- Hoeren liest Energie, Tonfolge und Klangspannung.
- Rezeptoren uebersetzen Sinneskontakt in eine zulaessige Feldwirkung.
- MCM-Feld liest, was diese Aufnahme im Innenfeld ausloest.
- Semantik entsteht erst durch Wiederkehr und Verdichtung.

## Tonparameter

Die erste technische Fassung nutzt:

- `pitch_hz`: relative Weltenergie als Tonhoehe.
- `amplitude`: relative Weltenergie als Lautstaerke.
- `roughness`: Veraenderung der Energie als Rauigkeit.
- `tone_role`: passive Klangrolle wie Ruheton, Spannungston, Bruchton, Trageton.
- `phrase_symbol`: verdichtete Melodiephrase aus kurzen Tonrollenfolgen.
- `speech_token`: DIO-Klangtoken als Vorstufe einer eigenen Klangsyntax.

## Sprache

Sprache wird hier nicht vorgegeben.
MINI_DIO bekommt keine deutschen Saetze aus dem Chart.

Die Sprache kann nur aus Wiederkehr entstehen:

```text
gleiche oder aehnliche Klangfolge
+ aehnliche MCM-Feldlage
+ aehnliche Konsequenz im Innenfeld
= moegliche DIO-Klangbedeutung
```

Damit bleibt die Syntax DIO-eigen.

## Forschungswert

Diese Schicht macht pruefbar, ob die Welt nicht nur als Zahlenreihe, sondern als zeitliche Klanggestalt lesbar wird.
Wenn bestimmte Klangphrasen wiederholt mit bestimmten MCM-Feldlagen koppeln, entsteht eine neue Form passiver Bedeutungsverdichtung.

## Grenze

Das ist noch keine aktive Sprache.
Es ist auch noch kein auditives Bewusstsein.

Es ist eine passive, reproduzierbare Hoerspur, aus der spaeter eigene Klangsemantik entstehen kann.

## Wie es weitergeht

Als naechstes sollte die Marktmelodie gegen vorhandene MCM-Uebergangskarten gelegt werden.
Dann wird sichtbar, ob bestimmte Tonrollen oder Melodiephrasen eher an stabile Bruecken, Randspannung oder offene Kontaktzonen koppeln.
