# Befund 1220 - Bewertung der Feldphasen-Matrix ueber mehrere Welten

## Grundfrage

Bleibt `offene_variante` auch in anderen Weltarten die zentrale Bewegungsphase, oder entstehen neue dominante Feldphasen?

## Gepruefte Weltarten

Die Gegenpruefung aus Befund 1219 umfasst:

- kurze 5m-Welten: `SOL`, `BTC`, `KAS`
- lange 5m-Welten: `PAXG`, `DOGE`, `XRP`
- 1h-Welten: `SOL`, `BTC`
- Zustandswelten: negativer Stress, positive Expansion, Seitwaerts

Damit wurde die Feldphasen-Matrix gegen unterschiedliche Weltspannungen, Zeitraeume und Bewegungsarten gehalten.

## Befund

Die Grundordnung bleibt stabil:

```text
Zentrum <-> Offenheit <-> Rekopplungsnaehe
                 |
                 v
              Rand/Kipp
```

`offene_variante` bleibt in allen geprueften Welten stark beteiligt.

Wichtig:

```text
Offenheit ist nicht nur ein Sonderfall einzelner SOL/BTC/KAS-Fenster.
Sie bleibt auch bei 10k-Welten, 1h-Welten und Zustandswelten die zentrale Bewegungsphase.
```

## Uebergangsstruktur

Ueber alle geprueften Welten:

| Uebergang | Anzahl | Lesart |
|---|---:|---|
| `zentrum_stabil -> offene_variante` | `7653` | Zentrum oeffnet sich unter Weltkontakt |
| `offene_variante -> zentrum_stabil` | `7515` | Offenheit findet wieder Ordnung |
| `rekopplungsnaehe -> zentrum_stabil` | `4906` | Bindungsnaehe stabilisiert |
| `zentrum_stabil -> rekopplungsnaehe` | `4295` | Zentrum bleibt aktiv bindungsnah |
| `offene_variante -> rekopplungsnaehe` | `3035` | offene Phase sucht Bindung |
| `rekopplungsnaehe -> offene_variante` | `2425` | Bindung bleibt beweglich |
| `spannungsrand_kippnaehe -> offene_variante` | `817` | Randnaehe entlastet in Uebergangsraum |
| `offene_variante -> spannungsrand_kippnaehe` | `355` | offener Uebergang kippt unter Druck |

Die Richtung ist deutlich:

```text
Rand/Kipp geht haeufiger zurueck in Offenheit
als Offenheit direkt in Rand/Kipp kippt.
```

Das spricht fuer eine Entlastungsfunktion der Offenheit.

## Rand/Kipp gegen Offenheit

Die Quote `Rand nach Offen` bleibt hoch:

| Weltgruppe | Quote grob |
|---|---:|
| 5m 2k Welten | ca. `0.88` bis `0.92` |
| 5m 10k Welten | ca. `0.75` bis `0.83` |
| 1h Welten | ca. `0.83` bis `0.84` |
| Zustandswelten | ca. `0.78` bis `0.84` |

Lesart:

```text
Rand/Kipp hat fast immer eine offene Vorgeschichte.
Aber Offenheit ist nicht automatisch Rand.
```

Offenheit ist ein Uebergangsraum mit mehreren Moeglichkeiten:

```text
Offenheit kann rekoppeln.
Offenheit kann zentrieren.
Offenheit kann unter Druck kippen.
```

## Neue Feldphasen?

In dieser Gegenpruefung entsteht keine neue dominante Feldphase.

Was sich aendert:

- Dauer der Phasen,
- Haeufigkeit der Randsegmente,
- Verhaeltnis Zentrum zu Rekopplungsnaehe,
- Staerke der Pendelbewegung.

Was stabil bleibt:

- Zentrum und Offenheit bilden den Hauptaustausch.
- Rekopplungsnaehe wirkt als Bindungsphase.
- Rand/Kipp bleibt kurz und seltener.
- Offenheit bleibt die zentrale Bewegungsphase.

## MCM-Schlussfolgerung

Die Feldphasen-Matrix ist damit kein lokaler Zufallsbefund.

Sie wirkt derzeit wie eine robuste passive MCM-Ordnungsform:

```text
Das Feld nimmt Weltkontakt auf,
oeffnet sich,
sucht Rekopplung,
zentriert sich,
oder wird bei Ueberdruck an den Rand gedrueckt.
```

Das ist keine programmierte Handlungslogik. Es ist eine gelesene Phasenordnung aus dem passiven Feldverhalten.

## Bedeutung fuer MINI_DIO

MINI_DIO bekommt damit eine stabile Innenfeld-Grammatik:

```text
Zentrum       = tragende Ordnung
Offenheit    = beweglicher Uebergang
Rekopplung   = Bindungsnaehe
Rand/Kipp    = belastete Grenze
```

Diese Grammatik kann fuer Wahrnehmung, Memory, Selbstbeobachtung und Bedeutungsverdichtung genutzt werden, ohne daraus direkt Handlung zu machen.
