# MCM-Drift Gegen Lokalkontakt

## Zweck

Diese Synthese verbindet:

- `857_MCM_VERDICHTUNGSZONEN_DRIFTLUPE`
- `858_MCM_VERDICHTUNGSZONEN_TOKENLUPE`
- `859_MCM_TOKEN_ROHWELT_SEGMENTLUPE`

Ziel ist zu klaeren, ob ein Zonenwechsel direkt aus dem lokalen Tokenkontakt erklaert werden kann oder ob eine hoehere Welt-/Feldbewegung beteiligt ist.

## Kernergebnis

Der Zonenwechsel ist nicht immer lokal erklaerbar.

Es gibt zwei Ebenen:

```text
lokaler Kontakt
  = was im konkreten Tokenfenster passiert

globale Zonenbewegung
  = wie sich derselbe Token ueber Weltgruppe, Weltbreite, Nachbarschaft und Rollenbindung verschiebt
```

Damit wird sichtbar:

```text
Ein Token kann lokal rekoppeln,
aber global trotzdem aus seiner bisherigen Verdichtungszone driften.
```

Das ist wichtig, weil es gegen eine zu einfache Lesart spricht.
MINI_DIO bildet nicht nur Einzelereignisse, sondern Feldbewegungen ueber mehrere Weltkontakte.

## Fachlicher Befund

Die neue Trennung lautet:

```text
Lokaler Kontakt erklaert die unmittelbare Feldlage.
Globale Zonenbewegung erklaert die Bedeutungsverschiebung.
```

Damit bekommt MINI_DIO eine sauberere MCM-Lesart:

- ein Kontakt kann tragen, ohne dass die Bedeutung langfristig stabil bleibt,
- eine Bedeutung kann reifen, ohne dass jeder lokale Kontakt stark positiv wirkt,
- Drift entsteht nicht zwingend aus einem einzelnen schlechten Kontakt,
- Rekopplung kann lokal und global zusammenfallen, muss es aber nicht.

## Bedeutung Fuer Die MCM

Das stuetzt die Annahme, dass ein MCM-Feld nicht nur punktuelle Reize speichert.
Es speichert bzw. verdichtet Feldbewegungen ueber Kontakt, Wiederkehr, Nachbarschaft und Weltbreite.

Kurz:

```text
MCM-Bedeutung = lokale Wirkung + zeitlich-raeumliche Feldbewegung
```

Das ist naeher an einer inneren Matrix als an einer einfachen Merkmalsliste.

## Wie es weitergeht

Als naechstes sollte fuer die vier Tokens eine Nachbarschaftslupe entstehen:

```text
Welche Tokens stehen davor?
Welche Tokens stehen danach?
Bleibt die Nachbarschaft stabil?
Kippt nur der Token oder das ganze lokale Feld?
```

Damit kann geprueft werden, ob Drift aus der Tokenqualitaet selbst entsteht oder aus einer veraenderten Nachbarschaft im MCM-Feld.
