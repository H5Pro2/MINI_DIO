Die frische Memory-Reproduktion wiederholt den vorherigen 4000er Befund exakt in der zentralen Matrix.
`1t5bcxp -> 183drjy` bleibt als Bewegung sichtbar und wird in der Reproduktionsmatrix dominant `eng_getragen` gelesen.

Damit ist die gerichtete Asymmetrie unter frischem Memory reproduziert:

```text
1t5bcxp -> 183drjy: eher eng getragen / rekoppelnd stabil
183drjy -> 1t5bcxp: eher fragmentiert / offen-driftend
```

Wichtig: Das ist keine starre Symbolbedeutung. Die gleiche Feldbewegung traegt je nach Weltlage weiterhin Nebenqualitaeten.
Der robuste Befund liegt in der Richtung der Bewegung und ihrer passiven Tragart, nicht in einem festen Einzelzeichen.

Fachliche Lesart:

```text
MINI_DIO reproduziert bei frischem Memory dieselbe gerichtete Tragart.
Die Innenfeldbewegung ist damit nicht nur eine einmalige Memory-Ablagerung.
Sie bleibt passive Wahrnehmung und darf nicht direkt in Handlung uebersetzt werden.
```
