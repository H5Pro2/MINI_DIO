# MCM-Feldmechanik: Funktionsableitung bis Befund 980

## Zweck

Diese Datei haelt die bisher ableitbaren Funktionen der MCM-Feldmechanik aus den MINI_DIO-Befunden fest.
Sie ist eine Forschungsnotiz, kein Beweisabschluss und keine Handlungslogik.

## 1. Das MCM-Feld bildet Ordnung ohne direkte Vorgabe

In den bisherigen Welten wurden wiederholt Feldformen sichtbar, ohne dass diese als Zielbild hart vorgegeben wurden:

- zentrumsnahe Bereiche,
- Randspannungsbereiche,
- offene Oberflaechen,
- Rekopplungsfelder,
- Bruecken- und Anschlussformen,
- Fragmentierungsoberflaechen.

Arbeitsableitung:

```text
Weltwirkung -> MCM-Feldreaktion -> passive Ordnungsform
```

## 2. Bedeutung ist nicht identisch mit Rolle

Ein Zeichen kann seine Rolle verlieren und trotzdem als Feldinformation sichtbar bleiben.

Beispiel:

```text
Bruecke verschwindet,
aber das Zeichen bleibt als Zentrum, Rand oder Rekopplungsfeld sichtbar.
```

Arbeitsformel:

```text
Rolle != Existenz
Bruecke != Bedeutung
Nicht-Bruecke != Chaos
```

## 3. Die MCM unterscheidet Weltqualitaeten

Mit `dio_frag_*` wurden verschiedene Oberflaechenqualitaeten lesbar:

- `fragmentierung_offen_randnah_jung`
- `fragmentierung_gemischt`
- `fragmentierung_randlastig`

Damit liest MINI_DIO nicht nur "viel" oder "wenig" Fragmentierung, sondern die Qualitaet der Feldoberflaeche.

## 4. Fragmentierung ist eine Feldreaktion

Die achte Welt zeigte extreme offene Jungfragmentierung.
Die neunte Stable-Welt zeigte eine kompaktere gemischte Oberflaeche.
Die zehnte Stresswelt zeigte randlastige Fragmentierung.

Arbeitsableitung:

```text
Fragmentierung ist kein starrer Systemzustand.
Fragmentierung ist eine weltabhaengige Oberflaechenreaktion des MCM-Feldes.
```

## 5. Das Feld zeigt passive Eigenregulation

Stress fuehrte nicht zu totalem Zerfall.
Stattdessen entstand eine kompakte randlastige Ordnung mit schwachem Zentrum als Gegenpol.

Arbeitsableitung:

```text
Welt wirkt ein.
Das Feld organisiert Spannung.
Die Oberflaeche bildet eine lesbare Klasse.
Bedeutung bleibt, wandert, fragmentiert oder rekoppelt.
```

## 6. Memory wird besser durch Rollenwechsel statt Loeschung

Fruehere Lesung:

```text
Zeichen nicht als Bruecke gefunden = Zeichen weg
```

Aktuelle Lesung:

```text
Zeichen nicht als Bruecke gefunden,
aber als Zentrum, Rand, Rekopplungsfeld oder Oberflaeche sichtbar.
```

Das fuehrt zu besserer passiver Memory-Qualitaet:

- weniger vorschnelles Verwerfen,
- weniger starres Auswendiglernen,
- bessere Trennung von Rollenverlust und Bedeutungsverlust,
- bessere Lesung von Weltwechseln.

## 7. Die MCM traegt mehrere passive Ordnungsarten

Derzeit sind mindestens diese passiven Ordnungsarten lesbar:

- Brueckenordnung,
- Rollenwechselordnung,
- Fragmentierungsordnung,
- Zentrumsordnung,
- Randspannungsordnung,
- Rekopplungsordnung,
- Oberflaechenordnung.

Diese Ordnungen sind keine Handlungsvorgaben.
Sie beschreiben, wie das Feld Weltwirkung innerlich organisiert.

## 8. Kernaussage

Die bisherige Forschung zeigt:

```text
Die MCM kann aus Weltwirkung innere Feldordnung, Bedeutungsspuren,
Rollenwechsel und Oberflaechenzustaende bilden.
```

Das ist der derzeit zentrale Befund der MINI_DIO-Feldmechanik.

## Offene Grenze

Die Befunde sind stark, aber noch Forschungsstand.
Sie zeigen wiederholbare Hinweise auf MCM-Feldorganisation, aber keine abgeschlossene Theorievalidierung.

Wichtig bleibt:

- weitere Reproduktion,
- weitere Welten,
- Stress-/Stable-/Expansion-Vergleich,
- Pruefung, ob Fragmentierung spaeter zu Rollenbindung reift.
